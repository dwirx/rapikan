## ✨ v1.1.2 — Optimasi Performa Instan `--ls` & Flag Opsional `--folder`

Pembaruan performa agar listing berkas berjalan sangat cepat tanpa lag secara default.

### ⚡ Peningkatan Performa:
- **Default Instan Tanpa Lag**: Saat Anda menjalankan `rapikan --ls`, program tidak lagi menghitung ukuran folder secara rekursif. Ini membuat program berjalan **seketika / instan** tanpa lag (kolom `Length` otomatis menampilkan `<DIR>` untuk folder).
- **Flag Opsional `--folder` (atau `-folder` / `--ls-folder`)**: Jika Anda ingin menghitung ukuran folder secara terperinci (rekursif), Anda dapat menambahkan flag `--folder` di bagian akhir perintah.

### 💻 Contoh Penggunaan Baru:
```bash
# Listing super cepat & instan (default)
rapikan --ls

# Listing dengan kalkulasi detail ukuran folder (opsional)
rapikan --ls --folder

# Rekursif instan
rapikan ./media --ls -r

# Rekursif dengan ukuran folder
rapikan ./media --ls -r --folder
```
