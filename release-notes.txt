## ✨ v1.1.0 — Kalkulasi & Tampilan Ukuran Folder pada `--ls`

Rilis ini membawa penyempurnaan besar pada fitur `--ls` (list berkas) dengan menghitung ukuran folder secara dinamis.

### 📁 Fitur Baru: Kalkulasi Ukuran Folder Otomatis
- **Tampilan Ukuran Folder**: Kolom `Length` kini tidak lagi menampilkan teks statis `<DIR>` untuk folder. Sebagai gantinya, `rapikan` akan **menghitung secara rekursif total ukuran seluruh isi folder** tersebut dan menampilkannya dengan satuan ukuran yang mudah dibaca (`B`, `KB`, `MB`, `GB`).
- **Peningkatan Performa Pintar**:
  - Untuk folder besar seperti `node_modules` atau `.git`, ukurannya tetap dihitung secara cepat pada sistem latar belakang, namun tidak akan diekspansi ke dalam pohon visual untuk menjaga kerapian terminal.
  - Untuk folder kosong (seperti folder `aku`), ukuran secara akurat akan ditampilkan sebagai `0 B`.
